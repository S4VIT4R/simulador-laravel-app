<b>1. El sistema cuenta con un inicio de sesión y una interfaz de registro</b>
<br>
<b>2. En el sistema se pueden dar de alta 2 tipos de usuarios:</b>
    <br>
<p>     2.1 Alumno </p>
    <br>
<p>     2.2 Docente</p>
